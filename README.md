# 📷 ASCII Cam

A real-time webcam-to-ASCII art converter running entirely in your browser. CRT monitor aesthetic with scanline effects, snapshot gallery, and multiple color modes.

## ✨ Features
- **Real-time ASCII** — Live webcam feed converted to ASCII characters at ~30fps
- **6 color modes** — Green (matrix), Amber, White, Cyan, Pink, Orange
- **5 character sets** — Full density, Medium, Light, Blocks (█▓▒░), Binary (10)
- **Resolution control** — Slider from low (60 cols) to high (160 cols)
- **Snapshot gallery** — Save up to 6 ASCII snapshots in the sidebar
- **Download** — Save ASCII art as a .txt file
- **Live stats** — FPS counter, columns, rows
- **CRT effects** — Scanline overlay + screen flicker animation
- **100% local** — Camera feed never leaves your device

## 🛠 Tech Stack
- Pure HTML, CSS, JavaScript
- Web APIs: `getUserMedia`, Canvas 2D, Clipboard
- Google Fonts: Share Tech Mono
- No external libraries

## 🎯 How to Use
1. Click **[ INITIALIZE CAMERA ]**
2. Allow camera access when prompted
3. Watch your face turn into ASCII art in real-time
4. Switch color modes and character sets in the sidebar
5. Click **[ SNAP ]** to save a snapshot to the gallery
6. Click **[ SAVE ]** to download as a .txt file
